About

Consentless is an open source, minimalist, and pricacy preserving traffic counter. It aims to be GDPR compliant without requiring consent. It does so by tracking only impressions on URLs and not IP address or other user identifiable information. Time is tracked only to minute precision to further reduce the ability to pinpoint users with supplemental data.

It can be installed on any VPS or hosted service.

It can be used as your sole tracking mechanism or it can be used as supplemental tracking for visitors that did not concent to your other tracking methods.

It's minimal and lightning fast. The client is four lines of JS while the server is under 50 lines of code.

Getting Started

There are three parts to counter: the client JavaScript, the server counter, and the reporting dashboard.

Client

Once you have the server configured you add the following script to the bottom of any web page to start tracking stats. This script goes just above the closing body tag.

<script src="https://counter.example.com/counter.js" async></script>

Server

Install the binary on a server, execute it, and redirect output to a file.
